Project Information

Group 8: John Russell, Frank Duffy, Guy Nasser Bssoume, and Jenna Murphy

For Project 2 our group has created an interactive dashboard built with Python Flask-powered API, HTML/CSS, JavaScript, and a SQLite database.

We have utilized Minneapolis Police Use of Force data which features over 33,000 records. Our dashboard filters the data by Minneapolis Neighborhood through a dropdown menu featured above. Three charts have been created to display the top 10 most common uses of forces, the top 10 most common problems that led to police use of force, and police use of force incidents over the years, spanning from January 2008 to July 2021.

Please take a moment to explore the data below.

Notes, Sources and Metrics:

Chart Library Source

The charts are powered by Chart.js. Chart.js is an open source HTML5 based JavaScript chart library which allows for simple yet flexible animated and interactive charts.

Data Source

Our dataset has been sourced through the City of Minneapolis located at opendata.minneapolismn.gov. The original data source is refreshed on a daily basis and dates back to January 2008.
